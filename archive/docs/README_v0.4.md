## 版本简介

Version_0.4 是社官网的第四个迭代版本，全面升级了主题切换系统。采用精美的太阳/月亮动画切换按钮，支持响应式亮色（米白）和暗色（深蓝）主题，具备 localStorage 持久化存储功能。

## 核心功能特性

### 主题系统

| 模式 | 背景 | 强调色 | 适用场景 |
|-----|------|-------|---------|
| **亮色** | 米白 `#f5f0e8` | 翠绿 `#27ae60` | 白天、光线充足 |
| **暗色** | 深蓝 `#0d1b2a` | 荧光绿 `#39ff14` | 夜间、护眼模式 |

### 切换按钮动画

- **太阳状态**：金色圆盘 + 旋转光芒射线
- **月亮状态**：银灰圆盘 + 陨石坑细节
- **星星闪烁**：5 颗四角星，随机闪烁动画
- **云层飘动**：2 朵白云，缓慢浮动效果
- **过渡动画**：0.5s 弹性缓动曲线

### 技术实现

| 技术点 | 说明 |
|-------|------|
| CSS 变量 | 全局主题变量，统一控制配色 |
| localStorage | 主题偏好持久化存储 |
| requestAnimationFrame | 流畅的状态切换 |
| 防闪烁机制 | 脚本初始化先于页面渲染 |
| 系统监听 | 自动适配系统主题变化 |
| 过渡锁 | 400ms 防止重复点击 |

## 使用说明

### 主题切换

1. 点击导航栏右侧的太阳/月亮按钮
2. 按钮会播放平滑的形变动画
3. 整个页面配色会随主题平滑过渡
4. 选择会自动保存，刷新后保持

### 系统适配

- 首次访问自动检测系统主题
- 手动切换后不再跟随系统
- 清除浏览器数据可重置为自动模式

### 无障碍

- 支持 `prefers-reduced-motion` 减少动画
- 按钮具备 `aria-pressed` 状态标识
- 高对比度文字确保可读性
