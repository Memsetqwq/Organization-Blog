## 版本简介

Version_0.2 是社官网的第二个迭代版本，采用全屏沉浸式设计理念。页面去除了传统的多栏布局，改为单页面分段展示模式，配合丰富的滚动渐入动画和现代化的卡片设计，提供更具冲击力的视觉体验。

## 核心功能特性

### 页面结构

| 区块 | 内容 |
|-----|------|
| **HERO** | 全屏欢迎区，动态背景画布，引导性按钮 |
| **ABOUT** | 社团数据展示，数字统计卡片 |
| **TECH** | 技术方向卡片，图片+描述+标签 |
| **EVENTS** | 活动展示，左右交替图文排版 |
| **CONTACT** | 联系方式与二维码 |

### 交互功能

- **滚动动画系统**：页面元素随滚动渐入显示
  - `an`: 向上渐入
  - `an-left`: 向右渐入
  - `an-right`: 向左渐入
  - `an-scale`: 缩放渐入
- **悬停效果**：按钮涟漪效果、卡片悬停浮起
- **导航高亮**：滚动时自动高亮当前章节
- **微信公众号二维码**：新增联系入口

### 视觉设计

| 设计元素 | 说明 |
|---------|------|
| 暗色主题 | 低调奢华的深色配色 |
| 毛玻璃导航栏 | `backdrop-filter: blur(20px)` |
| 渐变标题 | 文字渐变效果 |
| 浮动动画 | 滚动提示持续浮动 |

### 技术实现

| 技术点 | 说明 |
|-------|------|
| CSS 动画系统 | 自定义 cubic-bezier 缓动曲线 |
| 滚动驱动动画 | Intersection Observer API |
| 毛玻璃效果 | backdrop-filter 实现 |
| 响应式布局 | clamp() 流体排版 |
| Google Fonts | Inter 字体加载 |

## 使用说明

### 浏览页面

页面采用单页滚动设计，向下滚动即可依次浏览：
1. 欢迎区 → 点击按钮跳转 About 或滚动继续
2. 数据区 → 了解社团规模
3. 技术区 → 了解核心技术方向
4. 活动区 → 浏览近期活动
5. 联系区 → 获取联系方式

### 导航交互

- 导航栏固定在顶部，随时可跳转
- 滚动时导航链接自动高亮当前章节
- 点击链接平滑滚动至对应区块

### 移动端适配

页面已针对移动端优化，组件会自动堆叠排列。
