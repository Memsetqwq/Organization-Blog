## 版本简介

Version_0.3 是社官网的第三个迭代版本，基于 v0.2 进行了全面的精致优化。最显著的变化是引入了 **系统级明暗主题自适应**，页面会根据用户操作系统的配色方案自动切换深色/浅色模式。同时优化了 WebGL 背景效果和整体排版细节，提供更加精致的使用体验。

## 核心功能特性

### 页面结构

| 区块 | 内容 |
|-----|------|
| **HERO** | 全屏欢迎区，WebGL 动态背景，渐变遮罩过渡 |
| **ABOUT** | 社团数据统计卡片 |
| **TECH** | 技术方向展示卡片 |
| **EVENTS** | 近期活动图文展示 |
| **CONTACT** | 联系方式与二维码 |

### 核心升级

#### 系统级明暗主题

```css
/* 自动检测系统偏好 */
@media (prefers-color-scheme: light) {
  :root:not([data-theme="dark"]) {
    --bg: #f5f5f7;
    --text: #1d1d1f;
    /* ... */
  }
}
```

| 模式 | 说明 |
|-----|------|
| 自动模式 | 跟随操作系统设置 |
| 手动切换 | 点击按钮强制指定主题 |

#### WebGL 背景增强

- WebGL 画布仅覆盖 Hero 区域（性能优化）
- 底部渐变遮罩平滑过渡到内容区
- 明暗主题下颜色自动适配

### 交互功能

- **滚动动画**：元素渐入效果
- **主题切换**：自动/手动双模式
- **导航高亮**：滚动同步
- **按钮微交互**：涟漪效果、悬停反馈
- **微信公众号二维码**：联系入口

### 技术实现

| 技术点 | 说明 |
|-------|------|
| prefers-color-scheme | CSS 媒体查询实现系统主题检测 |
| 增强型 WebGL | 仅 Hero 区域渲染，降性能开销 |
| 主题变量系统 | 全局 CSS 变量统一管理 |
| 渐变遮罩 | 多层渐变实现平滑过渡 |
| Noto Sans SC | 优化中文显示 |

## 使用说明

### 主题模式

**自动模式（默认）**：
- 页面会自动读取操作系统的配色设置
- 在 macOS/Windows 深色模式下自动切换深色主题
- 在浅色模式下自动切换浅色主题

**手动模式**：
- 点击顶部导航的 **DARK/LIGHT** 按钮
- 会覆盖自动模式，强制使用指定主题

### 浏览页面

与 v0.2 类似，采用单页滚动设计：

1. **Hero** → WebGL 动态背景 + 欢迎语
2. **About** → 社团数据
3. **Tech** → 技术方向
4. **Events** → 近期活动
5. **Contact** → 联系方式

### 移动端适配

响应式布局，窄屏下自动堆叠为单列。
